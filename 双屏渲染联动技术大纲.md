# CesiumJS 双屏渲染联动技术大纲

## 1. 项目概述

基于 CesiumJS 实现同一页面内两个独立 Viewer 实例的双屏渲染，并在相机视角、时钟、图层、实体等方面实现双向联动。

## 2. 技术栈

| 组件 | 选型 | 版本 |
|------|------|------|
| 3D 引擎 | CesiumJS | 1.143 (CDN) |
| 影像服务 | Cesium Ion (默认影像) | - |
| 地形服务 | Cesium World Terrain | Ion Asset |
| 3D Tiles | Cesium OSM Buildings | Ion Asset #96188 |
| 前端框架 | 无（纯原生 HTML/CSS/JS） | - |

## 3. 核心架构

### 3.1 双 Viewer 实例

```
┌─────────────────────────────────────────┐
│              Browser Window              │
├───────────────────┬─────────────────────┤
│   Viewer A (3D)   │   Viewer B (2D/3D)  │
│   - SceneMode.3D  │   - 可切换 2D/3D    │
│   - 主控端        │   - 从属端          │
│   - 独立 Canvas   │   - 独立 Canvas     │
├───────────────────┴─────────────────────┤
│           Sync Controller               │
│   - Camera Sync (双向)                  │
│   - Clock Sync                          │
│   - Entity Sync                         │
│   - Layer Sync                          │
└─────────────────────────────────────────┘
```

### 3.2 关键限制

- CesiumJS **不支持**单 Canvas 多视口，必须创建两个独立 Viewer 实例
- Entity/Primitive 实例**不能跨 Viewer 共享**，需分别创建
- GPU 资源在两个 Canvas 之间完全独立，会产生双倍显存开销

## 4. 功能模块

### 4.1 双屏布局

- 左右分屏（默认 50/50），支持拖拽调整比例
- 响应式布局适配不同屏幕尺寸
- 可折叠/展开单侧视图

### 4.2 相机联动 (Camera Sync)

**核心实现方式：** 通过 `scene.preRender` 事件监听主视图相机变化，同步到从视图。

```javascript
// 双向联动核心逻辑
function syncCamera(source, target) {
  target.camera.setView({
    destination: source.camera.position.clone(),
    orientation: {
      heading: source.camera.heading,
      pitch: source.camera.pitch,
      roll: source.camera.roll
    }
  });
}
```

**防循环机制：** 使用标志位 `_syncing` 防止 A→B→A 的无限循环同步。

### 4.3 时钟联动 (Clock Sync)

- 共享同一 `Clock` 实例，或在 `tick` 事件中同步时间
- 播放/暂停/速度调节保持一致

### 4.4 图层联动 (Layer Sync)

- Imagery Layer 切换同步（卫星图、路线图等）
- 地形数据加载同步
- 3D Tiles 加载状态同步

### 4.5 实体联动 (Entity Sync)

- 在两个 Viewer 中创建相同实体（不共享实例）
- 实体的增删改操作双向同步
- 选中状态联动（点击 A 中实体，B 中高亮同一实体）

### 4.6 场景模式

- Viewer A：固定 3D 模式
- Viewer B：支持 3D / 2D / Columbus View 切换
- 模式切换时自动重新同步相机位置

## 5. 交互功能

### 5.1 独立控制

- 每个 Viewer 可独立缩放、旋转（解除联动时）
- 联动开关：可随时启用/禁用相机联动

### 5.2 标注与测量

- 双屏同步绘制点/线/面标注
- 距离/面积测量工具（结果在双屏同步显示）

### 5.3 飞行定位

- 点击预设地点，双屏同步飞行（`camera.flyTo`）
- 搜索定位联动

## 6. 性能优化

| 策略 | 说明 |
|------|------|
| `requestRenderMode` | 仅在场景变化时渲染，降低 GPU 负载 |
| 节流同步 | 相机同步使用节流（16ms），避免每帧双重计算 |
| 按需加载 | 非活跃 Viewer 降低 `maximumScreenSpaceError` |
| 共享数据源 | 实体数据在内存层面共享，仅渲染对象独立 |

## 7. 文件结构

